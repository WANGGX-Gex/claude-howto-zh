# 代码审查清单

## 安全清单
- [ ] 无硬编码的凭据或密钥
- [ ] 所有用户输入都有输入验证
- [ ] SQL 注入防护（参数化查询）
- [ ] 状态变更操作的 CSRF 防护
- [ ] 正确转义的 XSS 防护
- [ ] 受保护端点的认证检查
- [ ] 资源的授权检查
- [ ] 安全的密码哈希（bcrypt、argon2）
- [ ] 日志中无敏感数据
- [ ] 强制 HTTPS

## 性能清单
- [ ] 无 N+1 查询
- [ ] 适当使用索引
- [ ] 在有益的地方实现缓存
- [ ] 主线程无阻塞操作
- [ ] 正确使用 async/await
- [ ] 大数据集分页
- [ ] 数据库连接池化
- [ ] 正则表达式已优化
- [ ] 无不必要的对象创建
- [ ] 防止内存泄漏

## 质量清单
- [ ] 函数 < 50 行
- [ ] 变量命名清晰
- [ ] 无重复代码
- [ ] 正确的错误处理
- [ ] 注释解释"为什么"而非"做了什么"
- [ ] 生产环境无 console.log
- [ ] 类型检查（TypeScript/JSDoc）
- [ ] 遵循 SOLID 原则
- [ ] 正确应用设计模式
- [ ] 代码具有自文档性

## 测试清单
- [ ] 编写了单元测试
- [ ] 覆盖了边界情况
- [ ] 测试了错误场景
- [ ] 存在集成测试
- [ ] 覆盖率 > 80%
- [ ] 无不稳定测试
- [ ] 模拟了外部依赖
- [ ] 测试名称清晰
